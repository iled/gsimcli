############################################################################################
gsimcli: geostatistical simulation for the homogenisation  and interpolation of climate data
############################################################################################

***********************************
Welcome to gsimcli's documentation!
***********************************

gsimcli is a method and a software package to homogenise climate data.

This research project is hosted at `NOVA IMS <http://www.novaims.unl.pt>`_
(Lisbon, Portugal) and it is funded by the "Fundação para a Ciência e
Tecnologia" (`FCT <http://www.fct.pt>`_), Portugal, through the research project
PTDC/GEO-MET/4026/2012. See `approval and funding notice
<http://www.novaims.unl.pt/documentos/P_GSIMCLI_EN.pdf>`_.
